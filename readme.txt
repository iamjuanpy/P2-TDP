Readme 
------

Jugabilidad
-----------

*El numero de las casillas se cambia clickeando sobre la misma, avanzando de a 1.

*Se muestran las casillas en rojo en aquella fila/columna/bloque 3x3 
 que tenga numeros repetidos.

*Las casillas con el cuadro celeste son aquellas fijas, que no puede modificar el 
 jugador.

*El timer que comienza al principio de una partida puede llegar hasta 90 minutos,
 si no se ganó hasta ese entonces, se pierde y se abre el menu para jugar otra o 
 salir.

Aclaración
----------

*Para el funcionamiento correcto de la ejecución desde eclipse del codigo, la carpeta
 res con las imagenes y el tablero
 del juego debe estar dentro de la carpeta bin del proyecto.

*Para el funcionamiento correcto del ejecutable tiene que estar la carpeta res en el
 mismo directorio que el ejecutable, ahi dentro están las imagenes y el archivo del
 tablero (tablero2.txt).
